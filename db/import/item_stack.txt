//=============================================================
// 物品 - 叠加量限制数据库（自定义）
//=============================================================
// 根据所给类型限制道具的叠加量.
// 通常用于三转职业相关道具.
//
// 数据库格式:
// <道具ID>,<限制叠加量>,<类型>
//
// MaxStackAmount:
//	Stack limit for the item. Use 0 to disable a restriction.
//
// 类型值:
//  &1: 物品栏限制
//  &2: 手推车限制
//  &4: 玩家仓库限制
//  &8: 公会仓库限制
//
// 例如:
//  512,4,12  // 不允许在仓库中存放超过4个苹果.
